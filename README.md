# Grandpa's Website
https://jc2007xyz.github.io/grandpas-quotes-website/index.html 

Grandpa's Website is a responsive, multi-page website built with HTML, CSS, and JavaScript.
It celebrates my grandfather by preserving family stories, memorable quotes, personality traits, and his travels through an interactive map

## Overview
I created this project to document and preserve the stories, personality, and experiences that make my grandfather unique.
It also served as an opportunity to strengthen my web development skills, particularly in HTML and CSS.

## Features
- Interactive travel map
- Responsive design for desktop and mobile
- Collection of stories and quotes
- Custom CSS Styling
- Github Pages deployment
